### Дата
date +%y-%m-%d  
Из скрипта `` DATE=`date '+%y-%m-%d'` `` 


### Комманда как аргумент
`$(date '+%y-%m-%d')`  выполнит команду и вернет её р-т в stdout.


### Конкатенация
#### Переменная + Строка
```bash
var="myscript"
var=${var}.sh
echo $var
```
`myscript.sh`


### Условный оператор if

#### Если папка существует
```bash
if [ -d $FOLDER  ]; then
    # сделать что нибудь
fi
```


#### Если папка **НЕ** существует
```bash
if [ ! -d $FOLDER  ]; then
    echo "Папка не существует"
	exit 1
fi
```
